## What was done
- created an interactive dashboard using a .csv dataset sourced from Tableau Public
- analyzed data using a pivot table and aggregate functions (Max, Average) to calculate key player
dominance metrics, such as total weeks as world number one and average streak duration
- visualized performance trends through a dynamic column chart and applied conditional formatting to
highlight data, especially top-performing players
- implemented interactive slicers to enable dynamic data filtering by gender, date of birth and player
name.
